# MediPredict 🩺🤖

MediPredict is an AI-powered web application that predicts potential diseases based on user symptoms using Machine Learning algorithms.  
It combines **Machine Learning**, **Data Science**, and **Full Stack Web Development** to create an intelligent healthcare support system.


## 🚀 Key Idea

```text
User Symptoms → ML Model → Disease Prediction
```

The application allows users to select symptoms, receive AI-based disease predictions, and consult doctors online.



# ✨ Features

- 🔐 User Authentication (Signup & Login)
- 🩺 Symptom Selection System
- 🤖 AI-Based Disease Prediction
- 📋 Prediction Result Display
- 👨‍⚕️ Doctor Consultation Page
- 📅 Meeting/Appointment Booking
- 💬 Online Consultation Support



# 🛠️ Tech Stack

## Frontend Development

Frontend is built using:

- React.js
- Tailwind CSS

### Frontend Features

- Responsive UI
- Modern Design
- User Authentication Pages
- Disease Prediction Interface
- Doctor Consultation & Booking Pages


## Backend Development

Backend is built using:

- Node.js
- Express.js

### Backend Responsibilities

- API Creation
- Communication with ML Service
- Database Operations
- User Authentication Handling
- Prediction Request Processing


# 🧠 Machine Learning Integration

The ML model processes user symptoms and predicts possible diseases using trained machine learning algorithms.

### Workflow

1. User selects symptoms
2. Frontend sends request to backend API
3. Backend communicates with ML service
4. ML model predicts disease
5. Results are displayed to the user

# ⚙️ Installation & Setup

## 1️⃣ Clone Repository

```bash
git clone <your-repository-url>
cd MediPredict
```


## 2️⃣ Setup Frontend

```bash
cd frontend
npm install
npm run dev
```



## 3️⃣ Setup Backend

```bash
cd backend
npm install
npm start
```


## 4️⃣ Setup ML Model

Install required Python dependencies and run the ML service.

```bash
pip install -r requirements.txt
python app.py
```
